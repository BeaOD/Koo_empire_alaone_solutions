Creating an image carousel using CSS and JavaScript involves several steps. Here's a simple example to get you started: ### HTML ```html Image Carousel
``` ### CSS (styles.css) ```css body { margin: 0; padding: 0; box-sizing: border-box; } .carousel { position: relative; width: 80%; max-width: 600px; margin: auto; overflow: hidden; } .carousel-images { display: flex; transition: transform 0.5s ease; } .carousel-images img { width: 100%; flex-shrink: 0; } button { position: absolute; top: 50%; transform: translateY(-50%); background-color: rgba(0, 0, 0, 0.5); color: white; border: none; padding: 10px; cursor: pointer; } .prev { left: 0; } .next { right: 0; } ``` ### JavaScript (script.js) ```javascript let index = 0; function showSlide(n) { const slides = document.querySelectorAll('.carousel-images img'); const totalSlides = slides.length; if (n >= totalSlides) { index = 0; } else if (n < 0) { index = totalSlides - 1; } else { index = n; } const offset = -index * 100; document.querySelector('.carousel-images').style.transform = `translateX(${offset}%)`; } function nextSlide() { showSlide(index + 1); } function prevSlide() { showSlide(index - 1); } // Optional: Automatically change slides every 3 seconds setInterval(nextSlide, 3000); ``` ### Explanation - **HTML**: Sets up the structure of the carousel with images and navigation buttons. - **CSS**: Styles the carousel and positions the buttons. - **JavaScript**: Handles slide transitions and navigation functionality. Replace `image1.jpg`, `image2.jpg`, and `image3.jpg` with the paths to your images. Adjust the CSS and JavaScript as needed to fit your design and functionality requirements.
